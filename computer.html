Een computer is een comlpexe machine.
Hij werkt door een combinatie van 1 en 0 waarmee alles wordt gemaakt, er word dus gebruik gemaakt van een Binair stelsel
Een computer bestaat uit heel veel onderdelen zoals: processor, moederbord, grafische kaart, ram, opslag(ssd) en een koeler.
Om een computer opdrachten te kunnen geven moet je het een input geven. Dit kan bijvoorbeeld doormiddel van een muis, toetsenbord of een touchscreen.
Een processor in je computer ontvangt deze inputs en gaat ze verwerken. Deze verwerkte inputs worden output waardoor het bestuurprogamma weet wat de gebruiker wilt.
Outputs zorgen er ook voor dat de gebruiker dingen ziet op zijn/haar scherm.
